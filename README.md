#Resources No More!
Plugin to keep track of any rocks and trees that the player gets, leaving them permanently marked.
--
Model Outline Color - Changes the outline of any objects that have been depleted.
Tile Color - Changes the base tile color of objects that have been depleted.
Remove Rocks/Tree - Opts out of displaying removed rocks/trees. They still depleted, just not displayed.
Remove Tile Outline - Removes tile at the base of the objects.
Show Object Outline - Shows an outline around any depleted objects.
